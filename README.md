# Heart-Disease-prediction
Developed a predictive model using machine learning algorithms to identify probability of heart disease from structured healthcare data. Performed data cleaning, exploratory data analysis, and feature selection to improve model performance. The project demonstrates how AI can assist in early detection and risk assessment in the healthcare domain.

##TOOLS USED
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn (optional), Machine Learning
